# Pixhawk - NUC setting

## environment
    NUC : Ubuntu 18.0, ROS melodic
    Pixhawk : 1.8.2v
    HW : FTDI serial-to-USB Cable
  
## install Mavros in NUC
    sudo apt-get update
    sudo apt-get install ros-melodic-mavros ros-melodic-mavros-extras
    cd ~/catkin_ws
    catkin_make
    source devel/setup.bash
